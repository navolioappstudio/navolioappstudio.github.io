Lego Collezione — Supporto

Navolio App Studio · Versione 1.0 · iOS 17+

Domande frequenti

Come aggiungo un set alla collezione?
Tocca il pulsante rosso "+" in basso a destra nella schermata Collezione. Puoi aggiungere il set manualmente inserendo nome, numero, tema e stato, oppure usare lo scanner barcode per identificare il set dal codice sulla confezione.
Come funziona lo scanner barcode?
Tocca "+" → "Scanner". Punta la fotocamera verso il codice a barre sulla scatola del set. L'app legge il codice e pre-compila automaticamente il form. Completa i dettagli mancanti e salva.
Come cambio lo stato di un set (Montato, Smontato, Sigillato)?
Apri il dettaglio del set toccandoci sopra, poi tocca il badge "Stato" nella scheda. Si aprirà un picker per selezionare il nuovo stato. Salva con il pulsante in alto a destra.
Come aggiungo un set alla wishlist?
Apri il dettaglio di un set e tocca l'icona cuore in alto a destra. Il set viene spostato nella tab Wishlist. Per rimuoverlo dalla wishlist, tocca di nuovo l'icona cuore.
Come filtro i set per tema o stato?
Nella schermata Collezione, usa le chip filtro sotto la barra di ricerca per filtrare per stato (Montato, Smontato, Sigillato). Usa la barra di ricerca per cercare per nome, numero set o tema.
Come passo dalla visualizzazione a griglia a quella a lista?
Tocca l'icona in alto a destra nella barra di navigazione: l'icona con le righe passa alla lista, quella con i quadratini torna alla griglia.
Come elimino un set dalla collezione?
In modalità lista, scorri verso sinistra sulla riga del set per vedere il pulsante "Elimina". In alternativa, apri il dettaglio del set e usa l'opzione Elimina dal menu.
Come aggiungo foto al mio set?
Nel dettaglio del set, tocca l'area immagine in alto. Puoi scattare una foto con la fotocamera o scegliere un'immagine dalla libreria foto.
Il widget non si aggiorna, come faccio?
Il widget si aggiorna automaticamente ogni ora e ogni volta che apri l'app. Se continua a non aggiornarsi, tieni premuto sul widget → Modifica widget → conferma le impostazioni. Assicurati di avere iOS 17 o versione successiva.
Come ripristino il mio acquisto Premium su un altro dispositivo?
Vai su Impostazioni → Ripristina Acquisti. Assicurati di essere connesso con lo stesso Apple ID usato per l'acquisto originale.
Come elimino tutti i miei dati?
Elimina l'app da Impostazioni iOS → Generale → Spazio iPhone → Lego Collezione → Elimina App. Tutti i dati verranno rimossi definitivamente dal dispositivo.
L'app funziona offline?
Sì, completamente. Lego Collezione funziona al 100% offline. Non è richiesta connessione internet per nessuna funzionalità (eccetto l'acquisto Premium che richiede connessione per la validazione StoreKit).
